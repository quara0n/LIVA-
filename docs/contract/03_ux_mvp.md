# KONTRAKT – UX (MVP)

Overordnet:
- Én hovedskjerm
- Delt flate:
  - Venstre: Program (sluttprodukt)
  - Høyre: Øvelsesbibliotek (input)

Toppbar:
- Programtittel
- Status
- Én primær handling: Eksporter / Send

MVP-prinsipper:
- Programmet er alltid synlig og i sentrum
- Ingen skjult logikk: det som eksporteres må være eksplisitt redigerbart
- AI kan ikke “autofylle” uten at bruker ser og godkjenner
