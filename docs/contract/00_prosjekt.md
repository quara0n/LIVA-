
# KONTRAKT – LIVA Prosjekt

LIVA er et klinikerstyrt verktøy for å bygge trenings- og rehabiliteringsprogrammer.

Prinsipp:
- AI kan foreslå og strukturere, men kliniker bestemmer innhold og beslutninger.
- Produktets kjerne er: Program → redigering → eksport (PDF) → deling.

Ikke-mål:
- Diagnostikk
- Autonom programgenerering uten klinikerstyring
- Pasientapp i MVP
