# KONTRAKT – MVP scope

MVP skal levere:
- Programbygger (seksjoner + øvelser + dosering)
- Øvelsesbibliotek (velg øvelser)
- Validering som hindrer ugyldig program
- PDF-eksport av program

MVP skal ikke levere:
- Pasientlogin / pasientapp
- Tracking over tid, journal, progresjonshistorikk
- “Smart diagnose” eller automatiske kliniske beslutninger
